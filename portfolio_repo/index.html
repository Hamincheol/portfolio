Filterable TC
조건에 맞는 테스트케이스가 없습니다.

대표 Step 작성 방식

  1. 커스터마이징 화면에서 Skin 카테고리로 이동한다.
  2. 피부톤 프리셋을 선택하고 상세 보정 슬라이더를 조정한다.
  3. 다른 카테고리로 이동한 뒤 다시 Skin 카테고리로 복귀한다.
  4. 선택 상태와 미리보기 캐릭터 표현이 저장값과 일치하는지 확인한다.

Expected Result 작성 방식

  1. 선택한 피부톤이 캐릭터 미리보기에 즉시 반영된다.
  2. 카테고리 이동 후에도 선택 상태가 유지된다.
  3. 저장 후 재진입 시 저장값과 UI 선택값이 일치한다.
  4. 텍스처 누락, 메시 변형, 비정상 색상 왜곡이 발생하지 않는다.
